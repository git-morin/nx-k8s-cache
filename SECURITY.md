# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report them privately via [GitHub Security Advisories](https://github.com/git-morin/nx-k8s-cache/security/advisories/new).

Include as much detail as possible: affected component, steps to reproduce, potential impact, and any suggested fix. You can expect an acknowledgement within 48 hours and a status update within 7 days.
